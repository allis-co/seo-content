# 参考記事

100点に近いコンテンツの実例。記事の構造・深さ・読者への価値提供の仕方を参照する。

## 比較検討型の理想: リスティング広告代理店の選び方

- URL: https://liskul.com/listing-ads-agency-63553
- 著者: 長谷川智史（SO Technologies 執行役員CMO、業界歴20年）
- なぜ良いか:
  - 月額予算レンジ別に選択肢を分岐（30万未満/30〜100万/100〜1000万/1000万以上）
  - 代理店ビジネスモデルの経済的力学を構造で説明（手数料→担当社数→品質）
  - 「30万未満なら代理店に依頼しない」という選択肢を明示
  - 問い合わせ文面サンプルまで提供（読者の行動変容に直結するお土産）
  - 「自社がこの代理店にとって良い顧客か」という逆視点
  - 口コミサイトで従業員待遇をチェックするという独自視点

## KPI改善型の良例: Web広告のCPAを下げる9つの対策

- URL: https://moltsinc.co.jp/media/process/24167/
- 著者: 松尾謙吾（業界歴17年以上）
- なぜ良いか:
  - CPA = CPC ÷ CVR という構造を最初に提示し、改善の全体像を示す
  - 「必ず下がる施策は存在しない」という前提揺さぶり
  - 「CPA低下が常に正解ではない」（ビジネスの最終ゴールを踏まえて判断すべき）
  - 初心者→中上級者とレベル別に対策を分けている
  - 目標CPA自体がおかしいケースにも触れている

## 共通パターン（どの記事タイプでも応用可能）

1. いきなりハウツーに入らず、前提の構造・考え方を先に示す
2. 読者の状況で分岐させて自分ごと化させる
3. 「そもそもその前提合ってる？」と思い込みを揺さぶる
4. 一般論で止まらず具体的な判断軸を提示する
5. 並列で止まらず構造で提示し、優先順位を渡す
